Step 1: Creating an organization in GitHub

  1. Create an organization:
  2. Log in to GitHub.

  3. Select the "Create an organization" menu.

  4. Select a pricing plan:

  5. Choose an appropriate pricing plan for your organization.

  6. Set up the organization:

  7. Fill in the necessary settings.
  8. Confirm the creation.

➤ The URL of your future site will be generated from the "Organization name" field and will follow this structure: https://[site-name].github.io/.

Step 2: Create a new repository

  1. Create a repository: ➤ Create a repository on behalf of the organization.
    ➤ Select public/empty (the repository should not contain, for example, README.md).
  2. Clone the repository in PyCharm:
  3. Copy the repository URL (either HTTPS or SSH).
  4. In PyCharm, select "Clone Repository" or "Get from VCS".
  5. Copy the URL from your organization's page in GitHub (I choose SSH).
  6. Paste the URL and click "Clone".

➤ If you choose SSH, make sure you have SSH settings configured. We'll discuss why this is more convenient in later posts.
- Click “Trust Project” in the window that opens.

Step 3: Setting up a virtual environment

  1. Create a virtual environment:

  2. Run the following command in the PyCharm terminal: python -m venv .venv ➤ If you get a "command not found" error, check your installed Python version and adjust the command accordingly.
    Your Python version:
    For example, I used: ``` python3.13 -m venv .venv ➤ You can see the PyCharm version in the lower right corner of the screen.
    ➤ Setting up a virtual environment helps avoid conflicts between packages in different projects.
    ➤ It's not possible to install two or more different versions of the same package in one environment. The solution is to create another environment.

  3. Activate the environment:

  4. In the PyCharm terminal, run: bash source .venv/bin/activate

Step 4: Install MkDocs and the necessary packages

  1. Install via pip:
  2. Run the following in the terminal: bash pip install mkdocs mkdocs-material ➤ If needed, specify the version with: bash pip3.13 install mkdocs mkdocs-material
  3. Alternatively, use the "Packages" toolbar in PyCharm.
